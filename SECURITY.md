# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | ✅                 |

## Reporting a Vulnerability

GeoIntel is a security research tool. If you discover a vulnerability in the tool itself, please open a private security advisory at:

https://github.com/zanesense/geointel/security/advisories/new

Do not open a public issue for security-sensitive findings.

## Responsible Disclosure

We ask that you give 90 days to address any reported vulnerability before public disclosure. This applies to the GeoIntel codebase, not to the third-party services it queries.
